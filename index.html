Welcome to the DANAYI system. This is a minimalist card integration designed for seamless text embedding.

Each card like this one can be flipped to reveal additional information on the reverse side.

The cards support three size variants: compact, medium, and wide.

Right-click on any card to access the context menu with options to edit, delete, or change type.

All changes are automatically saved to your browser's local storage. To reset all data, use the reset button in the footer below.
